Чтобы использовать этот шаблон для проверки подлинности Windows Azure, см. статью по адресу: http://go.microsoft.com/fwlink/?LinkID=267940.

В противном случае, чтобы использовать этот шаблон для проверки подлинности Windows, см. инструкции ниже.

Размещение в IIS Express.
1. В обозревателе решений щелкните свой проект, чтобы выбрать его.
2. Если панель "Свойства" не открыта, откройте ее (F4).
3. В области "Свойства" для своего проекта выполните следующее.
   а) Для варианта "Анонимная проверка подлинности" задайте значение "Отключена".
   б) Для варианта "Проверка подлинности Windows" задайте значение "Включена".

Размещение в IIS 7 или более поздней версии.
1. Откройте диспетчер служб IIS и перейдите на свой веб-сайт.
2. В представлении "Просмотр возможностей" дважды щелкните пункт "Проверка подлинности".
3. На странице "Проверка подлинности" выберите "Проверка подлинности Windows". Если параметр "Проверка подлинности Windows
   недоступен, убедитесь, что на сервере установлены средства
   проверки подлинности Windows.

      Включение проверки подлинности Windows в операционной системе Windows.
      а) В панели управления выберите "Программы и компоненты".
      б) Щелкните "Включение и отключение компонентов Windows".
      c) Перейдите последовательно "Службы IIS > Службы Интернета > Безопасность"
         и убедитесь, что выбран узел проверки подлинности Windows.

      Включение проверки подлинности Windows в операционной системе Windows Server.
      a) В диспетчере сервера выберите пункт "Веб-сервер (IIS)" и нажмите кнопку "Добавить службы ролей".
      b) Перейдите последовательно "Веб-сервер > Безопасность"
         и убедитесь, что выбран узел проверки подлинности Windows.

4. На панели "Действия" щелкните пункт "Включить", чтобы включить проверку подлинности Windows.
5. На странице "Проверка подлинности" выберите "Анонимная проверка подлинности".
6. На панели "Действия" щелкните пункт "Отключить", чтобы отключить анонимную проверку подлинности.
